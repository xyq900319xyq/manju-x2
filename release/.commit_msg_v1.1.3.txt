v1.1.3 — 一键更新下载 UnicodeEncodeError 修复

## 修复
- 一键更新下载 Setup.exe 时报 UnicodeEncodeError: 'ascii' codec can't encode characters in position 51-54

## 根因
update.json 的 url 字段是 ".../releases/download/1.1.2/漫剧助手X-2_v1.1.2_Setup.exe",
Setup.exe 文件名含中文"漫剧助手"。_DownloadWorker 直接 urllib.request.Request(self._url) →
http.client.putrequest 内部 `request.encode('iso-8859-1')` 在中文 unicode 位置抛错。
4 层防御保证 url 全 ASCII + log 不爆。

## 改动
- core/updater.py _DownloadWorker.__init__:percent-encode url(中文 → %E6%BC%AB...),保证 urlopen 拿到的全 ASCII
- build_x2.py 写 update.json:url 字段 percent-encode,changelog_url 改纯英文 releases 页
- ui/main_window.py _on_error:info.html_url 段强制 ascii safe(中文 → '?')
- main.py 启动:sys.stdout/stderr.reconfigure(encoding='utf-8', errors='replace')
- 版本号 1.1.2 → 1.1.3 (main.py 2 处 + main_window.py 1 处 + .iss 1 处)
- docs/更新日志.md + RELEASE_NOTES_v1.1.3.md

## 验证
- urllib.parse.quote 含中文 url 后,request.encode('iso-8859-1') OK
- update.json url 字段全 ASCII(浏览器手动打开也能用)
