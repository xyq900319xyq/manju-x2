## 修复
- 分镜/提示词生成完 UI 不刷新:日志显示"完成: 分镜生成 #1" 但分镜面板一直"暂无分镜"、状态一直 pending,重启后才看得到

## 根因
main_window.py 剧集详情 tab 用了 id-only cache(self._episode_detail_cache[0] == ep.id),
persist 函数调 _show_episode_detail(ep_updated, ...) 时走 early return 复用旧 widget 串,
db 实际写好但 UI 没刷。
