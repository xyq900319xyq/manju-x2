# 魔因漫创 剧本导入格式完整参考

> 来源：`docs/SCRIPT_FORMAT_EXAMPLE.md` in the moyin-creator repo

## 格式元素一览

| 元素 | 格式 | 示例 |
|------|------|------|
| 标题 | `**《剧名》**` | `**《哥哥》**` |
| 大纲 | `**大纲：**` 一段话 | `**大纲：** 一个因童年自卑而逃离……` |
| 人物小传 | `角色名（年龄）：身份，性格` | `林星野（28）：创业失败者，自卑敏感` |
| 集标题 | `**第X集：标题**` | `**第一集：陨落与归途**` |
| 场景头 | `集-场 日/夜 内/外 地点` | `1-3 夜 外 新沪市` |
| 出场人物 | `人物：角色A、角色B` | `人物：林星野、苏晓` |
| 舞台指示 | `△` 前缀 | `△林星野推开玻璃门。` |
| 对白 | `角色名：台词` | `林星野：我不会再逃了。` |
| 表演提示 | `（括号内说明）` | `苏晓：（轻声）理论基础存在缺陷。` |
| 字幕/转场 | `【字幕：内容】` | `【字幕：三小时后】` |
| VO/画外音 | `角色名（VO）：` | `师尊（VO）：灵气不是玄学。` |
| SFX | `[SFX]：` 或转 △ | 保留原样或改为 △ |

## 完整示例片段

```
**《哥哥》**

**大纲：**
一个因童年自卑而逃离小镇的青年，在都市创业失败后，意外发现……

**人物小传：**
林星野（28）：创业失败者/火箭电车技术发现者，自卑敏感，执着坚韧
陆天擎（45）：天擎能源集团CEO，精明冷酷，掌控欲强

---

**第一集：陨落与归途**

---

**1-1 日 内 新沪市**
人物：林星野、苏晓

△林星野把最后一份文件塞进碎纸机。机器发出干涩的咀嚼声。

苏晓：（站在门口）工商的人下午来封门。

△林星野没回头。他盯着碎纸机出口吐出的白色条状物，像一地扯烂的绷带。

林星野：这次……

他嗓子发紧，咳了一声。

林星野：我不会再逃了。
```

## 剧本转换规则

当从影视剧本格式（`场x. 地点 - 日 - 内/外`）转换时：

1. **场景拆分**：每个 `场` → 独立场景块，编号 `集号-场号`
2. **场景头格式**：`{集}-{场} 日/夜 内/外 {地点名}`
3. **舞台指示**：所有动作/环境描述用 `△` 前缀
4. **对白格式**：`角色名：台词`（全角冒号）
5. **SFX 处理**：可保留 `[SFX]：` 或转为 `△` 描述
6. **人物标注**：每个场景块顶部加 `人物：` 行
7. **时间跨越**：用 `【字幕：】` 标注

## 注意事项

- 场景间用 `---` 分隔
- 角色 VO 用 `角色名（VO）：` 格式
- 集与集之间也用 `---` 分隔
- 文件保存为 UTF-8 TXT 格式
- 导入方式：剧本板块 → 粘贴全文或导入 TXT 文件
