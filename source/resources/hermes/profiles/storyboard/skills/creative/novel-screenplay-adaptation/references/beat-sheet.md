# Blake Snyder 15拍节拍表（Save the Cat! Beat Sheet）

## 概述

Blake Snyder 的节拍表是当前好莱坞最广泛使用的剧本结构工具。15个节拍覆盖一部电影的全部结构点。短剧同样适用，只需等比压缩时间轴。

## 15个节拍

### 1. 开场画面（Opening Image）— 0-1%
故事世界"之前"的状态。为终场画面建立对照。
- 展示主角的日常世界
- 暗示缺失或问题
- 短剧：1-2个镜头即可

### 2. 主题陈述（Theme Stated）— 5%
有人（通常是配角）说出故事的核心命题，但主角当时不理解。
- 以对话形式出现，不是说教
- 主角通常忽略或反驳
- 短剧：可以是一句台词或一个画面

### 3. 铺垫（Set-Up）— 1-10%
展示主角的缺点、局限、未完成的事。建立所有需要被"修复"的元素。
- 主角的缺陷
- 人物关系网
- 世界的规则
- 短剧：3-5个镜头快速建立

### 4. 催化剂（Catalyst）— 10%
打破平衡的事件。世界从此改变，无法回头。
- 必须是一个具体事件（电报、死亡、到来、发现）
- 不是"渐渐意识到"
- 短剧：通常是第一集的结尾钩子

### 5. 纠结（Debate）— 10-20%
主角在"去"与"不去"之间挣扎。这是人性时刻。
- 展示恐惧、犹豫
- 可能在多集中展开
- 短剧：1-2集

### 6. 第二幕开启（Break into Two）— 20%
主角做出不可逆的决定，进入新世界。
- 主动选择，不是被动卷入
- 物理上或心理上进入新世界
- 短剧：明确的"出发"场景

### 7. B故事（B Story）— 22%
副线，通常是情感/关系线。给故事注入心跳。
- 爱情线、友情线、师徒线
- 承载主题
- 短剧：可以是配角关系或情感暗线

### 8. 娱乐与游戏（Fun and Games）— 20-50%
故事的核心卖点，预告片的素材。主角在新世界中探索、学习。
- 这是"承诺的前提"被兑现的部分
- 轻松、有趣、展示世界观
- 短剧：最长的段落，3-5集

### 9. 中点（Midpoint）— 50%
假胜利或假失败。赌注翻倍。故事从"想要"转向"需要"。
- 假胜利："我们赢了！"——但代价即将显现
- 假失败："我们输了。"——但种子已种下
- 短剧：系列中段的重大转折

### 10. 反派逼近（Bad Guys Close In）— 50-75%
情况持续恶化。反派（内部/外部/两者）收紧包围圈。
- 团队内部分裂
- 敌人的反击
- 主角的缺陷开始反噬
- 短剧：持续施压，每集更糟

### 11. 一无所有（All Is Lost）— 75%
最黑暗的时刻。一切似乎都完了。
- 通常有一个"死亡的气息"（某人/某物/某种信念的死亡）
- 主角失去最重要的东西
- 短剧：系列中最绝望的一集

### 12. 灵魂黑夜（Dark Night of the Soul）— 75-85%
最低谷。主角面对真相，接受自己的缺陷。
- 这是"为什么我会在这里"的时刻
- 没有行动，只有反思
- 短剧：一个安静的反思镜头

### 13. 第三幕开启（Break into Three）— 85%
解决办法出现。主角看到了新的可能。
- 来自B故事/主题的启发
- 主角带着新的认识行动
- 短剧：反转的起点

### 14. 高潮/结局（Finale）— 85-99%
主角用新学到的东西解决问题。新旧自我的对决。
- 不再依赖旧方法
- 证明角色弧线的完成
- 短剧：最后1-2集的高密度冲突

### 15. 终场画面（Final Image）— 99-100%
与开场画面对照的新世界。证明改变已经发生。
- 镜像开场画面但完全不同
- 不是"他们从此幸福"，而是"世界变了"
- 短剧：最后一个镜头，与第一集第一镜呼应

## 短剧（每集1-2分钟）时间换算

| 长片（110页） | 短剧（10集×1分钟） |
|-------------|-----------------|
| 开场画面    | 第1集前5秒     |
| 催化剂      | 第1集结尾       |
| 纠结        | 第2集           |
| 第二幕开启  | 第3集开头       |
| B故事       | 第3集中段       |
| 娱乐与游戏  | 第3-5集         |
| 中点        | 第5集结尾       |
| 反派逼近    | 第6-7集         |
| 一无所有    | 第8集开头       |
| 灵魂黑夜    | 第8集中段       |
| 第三幕开启  | 第9集开头       |
| 高潮        | 第9-10集        |
| 终场画面    | 第10集结尾      |
