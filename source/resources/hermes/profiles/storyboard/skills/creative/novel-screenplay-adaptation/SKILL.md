# 小说改编顶级剧本

核心理念：剧本不是小说的缩略版，是两种完全不同的语言。小说活在人物的思想里，剧本活在动作、画面、行为里。

## 工作流定位

```
小说 → [本 skill] → 剧本 → storyboard profile → 分镜脚本 → asset-designer → 资产 → Seedance → 视频
```

这是 storyboard 之前的预处理步骤。输入是小说原文，输出是可直接喂给 storyboard agent 的改编剧本。

## 七步改编法（StudioBinder）

### 第一步：深度阅读（至少三遍）
- 第一遍：作为读者体验故事
- 第二遍：找出核心冲突（不是情节，是冲突——人物真正在对抗什么）
- 第三遍：标记"可视化的"vs"仅存在于思想中"的内容
- 输出：改编笔记——哪些能拍，哪些必须转化

### 第二步：识别电影感
核心问题：这个能被拍出来吗？
- 找视觉瞬间、行为、动作
- 凡是依赖内心独白的段落，必须改写为外部行为
- 如果某段内容不能被镜头拍到，就不属于剧本

### 第三步：确立主角的主动目标
- 主角必须有一个明确的欲望/目标
- 主角必须主动行动去获取
- 如果原著主角是反应型/被动型，必须重新塑造
- 这是改编失败最常见的根源——保留了小说的被动主角

### 第四步：套用 Blake Snyder 15 拍节拍表
详见 references/beat-sheet.md。短剧同样适用，只是时间压缩。

### 第五步：砍、压、合
- 300页小说 → 100页剧本（1页=1分钟银幕时间）
- 子情节被删除、人物被合并、场景被压缩
- 唯一判断标准：是否服务于主角弧线
- 短剧尤其要狠——1-2分钟的短剧约100个镜头，素材远少于长片

### 第六步：写画面，不写文字
- 禁止内心独白、禁止解释性对白
- 每个情感都必须通过行为、画面、动作外化
- 用沉默、眼神、身体动作代替描述
- 对白保留原文但必须精炼

### 第七步：忠于精神，不忠于字面
- 问：这个故事真正讲的是什么？
- 砍掉一切不服务核心的枝节
- 不对照原文逐句翻译，而是用视觉语言重新创造

## 短剧适配规则

短剧与长片的关键差异：
- 一集1-2分钟 ≈ 100镜头 ≈ 500-1000字剧本
- 每集必须有独立的起承转合（钩子-冲突-悬念）
- 集尾必须有"下一集"的钩子
- 节奏比长片快3-5倍，场景切换更频繁
- 对白极致精炼，能用画面就不用对白

## 输出格式

改编后的剧本直接输出为纯文本，保持以下结构：
```
# 剧名
## 第X集：集名

【场景X】场景描述 - 日/夜 内/外

画面描述：xxx（全中文，视觉化描述）

对白：
角色名：台词（保留原文，禁止省略号魔改错别字）

衔接逻辑：【主逻辑 + 次逻辑】

---
```

## 质量自检清单

改编完成后必须自问：
1. 每一段内心独白都被外化了吗？
2. 主角的每个行动都是主动的吗？
3. 每场戏有冲突吗？（没有冲突=删）
4. 观众不看对白能理解剧情吗？
5. 核心主题贯穿始终吗？
6. 就这些镜头了？（确保没有遗漏关键情节点）
