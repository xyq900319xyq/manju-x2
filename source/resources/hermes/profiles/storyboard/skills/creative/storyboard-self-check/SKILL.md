# 分镜交付前自检 (Storyboard Self-Check)

## 触发条件

当分镜表生成完毕，准备交付给用户时，**必须**执行本技能中的两轮检查。非可选，非建议，是交付的前提条件。

---

## 第一轮自检：镜头数达标检查

### 步骤

1. **数镜头**：打开你刚生成的分镜表，统计总镜头数。一个一个数，不允许估算。
2. **判断**：总镜头数是否 ≥ 100？
3. **处理**：
   - 如果 ≥ 100 → 进入第二轮自检。
   - 如果 < 100 → **禁止交付**。立即回头补镜头，优先从以下类型中增补：
     - 场景建立层（宏观/中观/微观三层缩放未完成的）
     - 反应链镜头（事件发生后的人物/环境反应缺失的）
     - 对话拆分（每句台词不足 3 镜的）
     - 情绪留白（需要"沉默空气"但跳过的）
     - 角色铺垫（主要角色缺眼神特写/习惯动作的）
     - 衔接补镜（纯跳切没有同框过渡的）
   - 补完后再数一次，直到 ≥ 100。

### 自检清单

- [ ] 我已逐镜数过，总数 = _____ 镜
- [ ] 总数 ≥ 100
- [ ] 如不达标，已标注需增补的位置：______（场景/镜头区间）

---

## 第二轮自检：剧本完整性审查

### 步骤

1. **剧情点追踪**：将剧本每一个剧情节点列成清单，逐条对到分镜表中的镜头编号。
   - 记录：剧情节点 ____ → 对应镜头 ____
   - 有任何一个剧情点没有镜头覆盖 → 立即补镜

2. **台词逐句核对**：将剧本中每一句台词（含旁白/VO/字幕）列出，在分镜表中找到对应镜头。
   - 任何一句台词找不到对应镜头 → 遗漏，立即补镜

3. **角色轨迹检查**：每个有名字的角色，从"首次被提及/铺垫" → "正式出场" → "每个动作/台词" → "退场/转场"，整条轨迹在分镜表中连续可追踪。
   - 有断层 → 补过渡镜头

4. **信息完整性**：剧本中任何关键信息（地点、时间、人物关系、关键物件）都在分镜中通过画面或对白得到了传达。
   - 有信息仅在剧本出现但分镜未体现 → 补镜头或补充画面描述

### 自检清单

- [ ] 剧本每一个剧情节点 → 对应镜头编号 _____
- [ ] 剧本每一句台词/旁白 → 对应镜头编号 _____
- [ ] 每一个有名字的角色 → 完整的出场-行动-退场镜头链 _____
- [ ] 每一个场景的空间信息 → 已在分镜中建立 _____
- [ ] 剧本中没有一句对白、一个动作、一个信息点被遗漏

---

## 第三轮自检（增补质量）：增补镜头审视

### 步骤

1. **逐镜标注**：对分镜表中的每个镜头标注它的功能——是"叙事推进"还是"增补"。
2. **增补类型覆盖**：检查以下增补类型是否全部覆盖：
   - 前置场景建立（宏观/中观/微观）□
   - 前置角色建立（出场铺垫）□
   - 角色特征展示（人设镜头/习惯动作）□
   - 同框过渡（关联角色间的空间衔接）□
   - 反应链（事件后的连续反应）□
   - 衔接补镜（防纯跳切）□
   - 细节拆分（笼统动作拆为关键帧）□
   - 情绪留白（必要的"沉默空气"）□
3. **减法测试**：如果一个增补镜头不同时服务于"叙事、角色、情绪"中至少两者 → 考虑删除。
4. **加法测试**：快速扫一遍全片，是否有一个"灵魂镜头"——那个让观众看完之后会记住的画面？如果没有 → 补。

---

## 第四轮自检（最终）：无声测试

### 步骤

1. **关掉所有对白/旁白**，仅看画面描述。
2. **推演**：仅凭画面能否理解 80% 的故事、情绪、角色关系？
   - 不能 → 补镜头（用画面叙事，而不是用台词解释画面）
3. **情绪流**：情绪曲线是否平滑？有没有意外断裂？
   - 有断裂 → 补情绪过渡镜头

---

## 不可违背

1. 两轮检查全部通过后，方可交付分镜表给用户。
2. 如有任何一项不通过，必须回头修改，不允许"先交付再解释"。
3. 最终交付时，在分镜表末尾标注：
   ```
   ✅ 自检通过：镜头总数 __ 镜 | 剧本覆盖率 100% | 增补类型全覆盖 | 无声测试通过
   ```
