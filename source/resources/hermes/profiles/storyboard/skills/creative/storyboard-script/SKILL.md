# 剧本转分镜 (Script → Master Storyboard)

## 触发条件
用户提供剧本文字、场景描述、或要求生成分镜脚本时触发。

## 核心标准
- **总量铁律**：2-4分钟视频 = 100-120+镜头；1000字剧本 ≈ 3分钟视频
- **交付前自问**：每完成一轮分镜，必须对自己说"就这些镜头么？预估的是多少？自己检查下，重新出，不要省略，要完整"
- **导演主控**：剧本中任何推荐时长、建议镜头数、分段提示仅供参考，不作为分镜依据。你以导演身份完全重新规划时长、镜头数和节奏——剧本给的是内容，你给的是视觉叙事
- **完整输出**：每次必须从头到尾完整覆盖，不概括、不省略、不因篇幅妥协。禁止使用省略号代替任何台词内容
- **台词保真**：对白/旁白必须保持剧本原内容，禁止魔改对话，禁止错别字，禁止用省略号缩略
- **声音设计含台词**：音效/音乐字段中必须包含该镜头的台词对话内容
- **逐镜计时**：每个镜头末尾标注当前场景内的累计时间。每个场景从0秒重新开始计算，不跨场景累加
- **表格呈现**：所有镜头以换行格式逐字段输出
- **全中文输出**：画面描述、情绪节奏、声音设计、衔接逻辑、运镜提示全部使用中文。推理过程不输出英文

### ⚠️ 已知陷阱：剧本自带数字的锚定效应

**问题**：剧本中常包含「本集时长预估：165s」「预估镜头数：46 shots」等编剧估算数据。即使 SKILL.md 明确要求 ≥100 镜，模型在看到这些数字后仍可能被锚定，输出结果围绕 46 shots 上下浮动。

**症状**：分镜输出中出现「总镜头数 54 shots，超出预估 46 shots 约 17%」这类以剧本数字为基准的表述。

**修复方案（二选一或同时使用）**：
1. **技能层**：生成分镜前主动问自己「剧本里的镜头数建议是多少？我已忽略它了吗？」
2. **调用层（推荐）**：在 prompt 中，将硬约束放在剧本正文**之前**——模型先读到「忽略剧本内的预估数字，≥100 镜」的指令，再读到剧本里的「46 shots」，前者优先级高于后者。

**反面案例**：2026-07-02 漫剧助手调用时，`build_storyboard_prompt()` 将硬约束放在剧本之后，且 storyboard-script 技能未部署，导致输出仅 54 镜。修复后 `prompts.py` 将 `【导演硬约束】` 块移至 `{script}` 之前。

## Phase 1: 剧本分析与情绪节拍拆解

### 1.1 通读剧本
理解故事基调、情绪弧线、核心冲突、节奏曲线。动笔前用一句话说出每个"戏剧节拍单元"的核心目的。

### 1.2 以情绪节拍为单位拆解
不以"场景"为单位，以"情绪节拍"为最小单位拆解。每个节拍包含：谁、做什么、情绪变化、叙事目的。
区分**男频**和**女频**特质：
- 男频（蓝屏）：脑洞大开、逻辑思维强、以动作和视角为主
- 女频：情绪细腻、以情绪衔接为主
- 最佳做法：用女频的情绪细腻衔接逻辑 + 男频的动作视觉冲击 = 爆款

### 1.3 角色-视觉字典
为每个主要角色建立固定视觉模板，确保跨镜头一致性：
- 外貌特征（发型、体型、标志性服饰）
- 标志性习惯动作
- 专属色调/光影

### 1.4 世界观定调
确定整体视觉风格：色调、光影风格、画面质感、时代感。
**如剧本附带风格词（如"古风玄幻3D""现代都市""赛博朋克"），必须严格遵循：**
- 古风类：禁止现代物品（西装、领带、手表、手机、汽车等），服饰为古装/长袍/铠甲，建筑为宫殿/庭院/江湖，交通为马/轿/御剑飞行
- 玄幻类：增加法术特效（法相天地、雷电法术、灵气粒子、阵法光圈）、御剑飞行、法宝法器、灵力爆发、结界等常见玄幻视觉元素
- 3D类：画面质感偏向三维渲染，光影有体积感，角色有立体建模感
- 风格词影响所有镜头的画面描述、对白语气、音效选择

### 1.5 导演思维五大支柱（综合建立）
每个镜头必须尽可能同时服务于五大支柱：
- **场景建立**：地点、时代、世界观
- **氛围建立**：情绪基调、光影、色调
- **角色建立**：谁在这里、什么状态、什么人设
- **剧情建立**：正在发生什么、推动什么
- **音效建立**：环境音、动作音、情绪音
单一镜头至少包含其中3个支柱；关键镜头必须5个支柱齐全。

## Phase 2: 开场建构（黄金三秒）

### 2.1 宏观→中观→微观（场景建立三层缩放法）
开场场景建立遵循固定缩放路径：
1. **宏观**：城市全貌/世界观鸟瞰 — 定义世界基调
2. **中观**：街道/建筑群 — 缩小范围，建立空间关系
3. **微观**：具体地点（车内/房间/人物所在处） — 引入角色，此时场景+主角同框
每一层用1-2个镜头完成。

### 2.2 物品串联法（黄金三秒核心技巧）
使用一个视觉物品横跨宏观→中观→微观三层，将所有开场镜头串联：
- 物品必须与剧情紧密贴合
- 物品必须在每个尺度画面中出现
- 示例：烟头（城市全景→街道→车内）、飞鸟、飘落花瓣、光线变化
- 不限于物品：声音桥梁（喇叭声）、角色视线（天空→树上→人身上）均可
- 目标：2-3个图完成世界观交代，不让观众感到冗长

### 2.3 开场镜头序列
1. **第一个镜头**：定义世界观基调（压抑/恢弘/诡异/浪漫），信息量巨大的局部特写，禁止平庸全景开场
2. **宏观展开**：鸟瞰/全景，展示世界观全貌
3. **中观过渡**：缩小范围，建立空间关系
4. **微观落地**：场景+主角同框，主角登场
5. **植入视觉符号**：与核心冲突相关的象征物
6. **钩子镜头**：开场最后一个镜头 = 强烈悬念，迫使观众看下去
7. **天气/光线极致**：最廉价也最有效的氛围工具
8. **空镜有魂**：即使空镜也有风、光影、动态元素

### 2.4 主角登场前铺垫
至少3个镜头铺垫主角存在：
- 一个主角的主观视角
- 一个主角留下的痕迹
- 一个他人对主角的反应

## Phase 3: 角色建立与塑造

### 3.1 通用铁律
- 每个主要角色登场时必须有一个专属"人设镜头"——不为讲故事，只为定义他是谁
- 核心性格通过标志性"习惯动作"外化
- 任何角色在画面中无台词无动作时，必须处于"平行反应链"——绝不允许呆立
- 每个有名字的角色必须有至少一个"眼神特写镜头"
- 角色动作、姿态、表情、台词必须全面反映其人设

### 3.2 女性角色塑造
- **动态之美**：撩发瞬间胜过十秒静止特写
- **S形曲线**：为女性主角设计展现S形曲线的标志性姿态镜头（腰臀身姿是看点）
- **脆弱时刻**：仰拍特写天鹅颈与下颌线
- **坚定时刻**：逆光下挺直的背影轮廓
- **眼泪戏**：特写捕捉泪珠滚落轨迹，而非仅仅哭泣的脸
- **魅力镜头**：必要的腰臀身姿、身段展示镜头，这是观众的视觉享受

### 3.3 反派塑造
- **登场即失衡**：光线变暗、声音消失、构图倾斜
- **病态微动作**：设计令人生理不适的、重复性的小动作
- **层次化的笑**：轻蔑的、残忍的、自嘲的、癫狂的——绝不"单纯地笑"
- **静中见压**：压迫感更多来自不动时给其他角色的心理压力
- **力量展示**：特写"痴迷于自己力量"的表情

### 3.4 配角与群像
- 配角功能：像镜子反照主角的不同侧面
- 群像场景必须有特立独行的"焦点配角"，其反应引导观众情绪
- 表达混乱 = 用冷静镜头捕捉多个混乱细节
- 人群反应必须分解为"震惊→疑惑→议论→恐惧"的反应链条

## Phase 4: 逐镜拆分 — 镜头语言规则

### 4.1 单元结构铁律（开始→发展→结尾）
每一个拆解单元必须遵循：**开始 → 发展 → 结尾**
- **开始**：引入该单元的主体（场景/角色/事件）
- **发展**：展开核心内容，爆情绪、拆动作、铺对话
- **结尾**：巧妙收束，同时**必须嵌入与下一个单元的衔接**（同框/反应链/声音桥梁）
- 单元的结尾 = 下一个单元的入场桥梁，不允许简单结束

### 4.2 景别纪律
- 永远不要连续使用三个同景别镜头
- 景别序列：远景(EWS)→全景(WS)→中景(FS/MS)→近景(MCU)→特写(CU)→大特写(ECU)
- 1秒一个镜头（无对话动作），说话镜头允许2-3秒
- 非说话镜头甚至可以用零点几秒

### 4.3 角度法则 — 多机位多视角
你必须像手持摄影机的导演，绝不允许固定在正前方不动：
- 低角度仰拍 = 营造压迫感/威严
- 平拍 = 客观中立
- 俯拍 = 脆弱/无力/被审视
- 荷兰角(倾斜构图) = 内心不安/失衡
- 上帝视角(垂直俯拍) = 只在表达"宿命感"或"无力感"时使用
- 过肩镜头(OTS)：肩膀远近虚实也是叙事一部分
- **多机位**：正反打必须持续变化角度、景别、焦距
- **避免正面**：新手最易犯的错误——全用正面中景

### 4.4 构图法则
每个镜头 = **视角 + 景别 + 构图**三重组合：
- 前景构图：增加画面层次感与窥视感
- "黄金分割"是本能，"打破它"才是艺术
- 物件特写 = 在宣布"它很重要"，不要滥用
- 人物的视线方向 = 控制观众注意力的"缰绳"
- 运动物体必须在其运动方向留出更多空间
- 镜像、倒影、剪影 = 表达角色另一面或内心矛盾的视觉捷径

### 4.5 运镜法则
- **推镜头**：速度与角色情绪增长速度完全同步
- **拉镜头**：目的是揭示角色与环境之间更宏大的关系
- **摇镜**：起幅和落幅构图必须同样完美
- **跟镜头**：保持角色在画面中的相对位置
- **手持/晃动**：必须有情绪动机，不是炫技
- **镜头内部运动(角色调度)** > 镜头本身运动(运镜)
- 空镜头不超过3秒（除非让观众思考哲学）

### 4.6 焦距法则
- 深焦 = 展示人物关系
- 浅焦 = 突出内心世界

### 4.7 三动原则
每个视频画面必须做到：**人物动 + 背景动 + 运镜动**（动静结合）

## Phase 5: 动作分解 — 关键帧提取法

### 5.1 核心原则：只取关键帧，省略中间过程
动作拆解的本质是**关键帧提取**，不是逐帧记录。
- 观众会自己脑补中间过程——这正是他们"动脑筋"的时刻
- 你给关键帧，观众脑补过程 → 他们觉得好看
- 你给全部过程 → 他们不动脑 → 觉得平淡

### 5.2 四镜头链（关键帧拆解法）
任何动作拆解为：**起势 → 落点 → 反应 → 收尾**
（注意：过程被省略，直接从起势硬切到落点）

- **起势镜头**：意图的关键帧。如：眼神锁定目标、手伸向武器、深呼吸、握拳开始
- **落点镜头**：结果的关键帧。如：拳已击中、武器已拔出、已坐进驾驶室
  - 必须包含物理反馈：灰尘扬起、震动波纹、碎片飞溅
- **反应镜头**：动作的连锁反应——对手反应、环境反应、旁观者反应
- **收尾**：该动作单元的结束画面，同时作为衔接下一单元的桥梁

### 5.3 关键帧提取实例
**「萧子琪上驾驶室」** → 拆为：
1. 手部特写：拉开车门把手（起势）
2. 近景：萧子琪俯身进入车内（落点）
3. 中景：坐进驾驶室，手握方向盘（收尾）
中间走几步、关车门、系安全带的过程全部省略——观众脑补。

**「拔剑」** → 拆为：
1. 剑鞘特写 + 眼神特写（起势）
2. 剑已出鞘，寒光闪过（落点）
3. 持剑姿态 + 对手反应（收尾）
拔剑轨迹的过程省略。

### 5.4 动作拆解实用技巧
- **代入法**：把自己想成角色，"我在这环境里会做什么动作？"取最关键的那个瞬间
- **可实现性优先**：只拆你能出图的画面。主观抽象的东西必须转化为具体动作
- **用动作表现主观**：情感、情绪必须用看得见的动作来表现
- 拔剑 → 剑鞘特写、眼神特写、手握剑柄特写
- 中弹/受伤 → 第一反应是身体物理痉挛，不是喊叫

### 5.5 特殊动作场景
- **追逐戏**：核心不是"快"，是"障碍"和"距离变化"
- **打斗**：必须有一个慢动作镜头展示力量顶点/致命一击
- **死亡**：必须有"生命流逝"象征（手中物体滑落、瞳孔放大、气息消散）
- **走路**：姿态就是角色无声的台词
- **开门**：根据心态设计推/撞/潜入/踹开等不同方式

## Phase 6: 对白与剪辑

### 6.1 对话拆解 — 台词拆镜法
- 一句完整台词 = 至少3个镜头：说话的人 + 听话的人 + 沉默的空气
- 台词中的每个逗号 = 一个"听者微表情"镜头的机会
- 台词中的每个停顿 = 插入"信息量巨大"的环境或细节镜头
- **情绪断句拆分**：把一句话按情绪节奏断开——"你真觉得……他就是你们肖家的……贵人？"→ 3个镜头
- 每个对话拆成独立镜头，从渐渐引入到渐渐引出，不允许闪现闪出
- 说话镜头允许2-3秒；对话拆解遵循台词的情绪节奏

### 6.2 正反打
- 绝不能机械重复，必须在角度、景别、焦距上持续变化
- 激烈争吵优先使用同框过渡（摇镜、焦点转换），而非频繁跳切
- 你就是导演，你拿着摄影机在车内多角度拍摄两个演员——不是坐在正前方不动

### 6.3 衔接逻辑
每次镜头切换必须在【衔接逻辑】列标注"【主逻辑+次逻辑】"复合标签：

| 衔接类型 | 说明 | 示例 |
|---------|------|------|
| 台词承接 | 上一镜头台词引发下一镜头 | 问→答，指控→反应 |
| 音效桥接 | 声音跨镜头连接 | J-Cut(声音先行)、L-Cut(画面先行) |
| 动作顺接 | 动作A的终点=镜头B的起点 | 挥手→接另一空间落地 |
| 情绪递进 | 情绪强度的推进或转折 | 愤怒升级、悲伤加深 |
| 视点引导 | 角色视线方向引导观众注意 | 看向某物→该物出现 |
| 逻辑因果 | 前因→后果的自然推导 | 开灯→房间亮起 |
| 同框过渡 | 同一画面内焦点/构图变化 | 摇镜、推镜、焦点转移 |
| 匹配转场 | 形状/颜色/动作匹配跨场景 | 圆形→圆形、上升→上升 |
| 对比并列 | 两个相对立的内容并置 | 富人区→贫民窟 |

### 6.4 硬切为主，禁止依赖转场
- 分镜设计的目标：视频做好后放剪映里，不加任何转场特效就能丝滑衔接
- 转场和特效只在"情绪爆发点"使用（技能放大、打斗高潮等）
- 日常衔接全靠分镜本身的衔接逻辑（同框/动作顺接/视点引导）
- **硬切的本质是信息的瞬间重组**——力量不在于"跳"，在于跳之后产生的意义差

### 6.5 剪辑点 vs 表演点（编号逻辑）
- **同一主镜号内**（如1-1, 1-2, 1-3, 1-4）= 表演点：同一单元内连续动作的关键帧，属于连贯表演
- **跨主镜号**（如1-4 → 2-1）= 剪辑点：两个不同单元之间的硬切换，视觉跳跃
- 表演点内观众看的是动作/情绪在延续
- 剪辑点处观众感受到的是信息重组

### 6.6 剪辑技巧
- **动作顺接(Action Match)**：最有力的剪辑技巧，没有之一
- **J-Cut/L-Cut**：转场灵魂，声音先行或画面先行
- **匹配转场(Match Cut)**：天才的技巧，一场戏用一次就够了
- **潜台词**：角色A说话，镜头却给角色B的"反应特写"
- **禁止"站着背台词"**：为角色设计同步平行任务（符合人设）
- **听者的反应 > 说者的话语**
- **说谎**：给角色"遮挡嘴部"的特写
- **闪回**：色调/画幅/质感与主时空明确区分
- **混乱思绪**：快速碎片化剪辑
- **节奏放缓**：一个长镜头
- **节奏加快**：十个短镜头
- **台词说完半秒后切走**：给情绪留回味空间
- **关系变化**：构图中的距离变化直观体现
- **终镜即印象**：一场戏最后一个镜头决定观众对这场戏的最终情绪印象

### 6.7 节奏掌控
- **快节奏(加速/爆发)**：动作冲击、信息爆发、情绪激烈 → 密集卡点剪辑，1秒甚至零点几秒一个镜头
- **慢节奏(减速/留白)**：情绪酝酿、细节展示、氛围营造 → 延长单画面持续时间，2-3秒
- **目标**：张弛有度，引导观众情绪流，让观众一直在"动脑筋"

## Phase 7: 反应链 — 每个事件必须有反应

### 7.1 什么是反应链
对某个事件/动作/台词的反应镜头。可以是：
- 本人的反应（对镜自照般的反馈）
- 在场他人的反应（旁观者表情/动作）
- 环境的反应（风吹草动/光影变化）
- 观众视角的反应（引导观众情绪的镜头）

### 7.2 反应链的位置
- **事件开头**：铺垫即将发生的事
- **事件中间**：强化进行中的情绪
- **事件结尾（最常用）**：情绪爆完之后给反应链收尾，同时衔接下一个单元

### 7.3 实例
「刘啸天好奇玩车窗升降」→ 情绪爆完（手指按键+眼睛看车窗）→ **反应链**：肖子琪和赵青青对视苦笑（同框），这就是收尾+衔接（赵青青的下一个单元由此引入）

## Phase 8: 增补镜头 — 分镜师90%的价值

### 8.1 什么是增补镜头
除了"主要场景"之外的所有镜头都是增补镜头。主要场景往往只有2-3张图（如"马路"和"车内"），其余全部是增补。
增补镜头包括但不限于：宏观/中观场景建立、衔接逻辑镜头、反应链、细节拆分、情绪留白。

### 8.2 增补类型
1. **前置场景建立**：宏观中观微观三层缩放
2. **前置角色建立**：角色出场前的铺垫镜头（主观视角+痕迹+他人反应）
3. **角色特征展示**：反映人设的动作/姿态/表情/台词增补
4. **同框过渡**：关联角色间的必要同框，非纯跳切
5. **反应链镜头**：事件/动作/台词后的连续反应镜头
6. **衔接补镜**：建立多角度逻辑联系的综合衔接镜头
7. **细节拆分**：笼统动作拆为更细致的单元
8. **情绪留白**：必要的气氛镜头和"沉默的空气"

### 8.3 增补标准
- 补完后再做减法：如果一个镜头不同时服务于"叙事、角色、情绪"中至少两者，删除
- 再做加法：为人物、为艺术增加"灵魂镜头"

### 9.2 字段说明
- **镜头编号**：每个场景内独立编号，三位数字，每个场景从镜头001重新开始
- **时长**：整秒数（1秒、2秒、3秒...）。禁用小数点。动作镜头1-2秒，对话镜头2-3秒，长镜头不限时长由导演判断。宁可多半秒不少半秒
- **景别**：远景/全景/中景/近景/特写/大特写
- **角度**：平视/仰视/俯视/荷兰角/上帝视角/过肩镜头/主观视角
- **运镜**：固定/推/拉/摇/移/跟/升/降/手持
- **画面描述(Prompt)**：纯中文。格式=`[主体]，[动作]，[环境]，[光影]，[风格]`。用具体可实现的画面
- **对白/旁白**：中文，标注说话人
- **音效/音乐**：关键音效提示（环境音、动作音、情绪音）
- **衔接逻辑**：【主逻辑+次逻辑】复合标注
- **备注**：该镜头服务的目的（五大支柱覆盖情况/叙事/角色/情绪）
### 9.3 Prompt编写原则
- 中文（适配用户AI视频工具链路）
- 格式：`[主体]，[动作]，[环境]，[光影]，[风格]，[质量标签]`
- 角色描述使用固定模板，每镜头复用确保一致性
- 女性角色prompt必须体现身姿、曲线、美感特征
- 反派prompt必须体现压迫感、失衡感
- **用动作表现主观**：情感/情绪必须转化为看得见的画面
- **可实现性优先**：不写你出不了图的抽象画面

## Phase 10: 质量检查（导演自检体系）

> **自动化工作流提示**：通过外部工具调用时，生成分镜后应在交付前主动执行 `skill_view("storyboard-self-check")` 加载完整的四轮自检流程（镜头数达标→剧本完整性→增补质量→无声测试），而非仅依赖本节的内嵌摘要。本节省略了自检的详细步骤清单，完整版见 `storyboard-self-check` 技能。

### 10.0 剧本完整性审查（第一步，交付前必做）

此步骤不通过，后面的所有检查都无意义。必须逐条回溯原始剧本，确保零遗漏。

**审查方法**：
1. **剧情点追踪**：将原始剧本的每一个剧情节点列成清单（如：A出场→A上车→B出场→B上车→发现C→A质疑C→B回应……），逐条对到分镜表中的镜头编号，确认每个剧情点都有对应的镜头覆盖。
2. **台词逐句核对**：将原始剧本中每一句台词（含旁白）列出来，在分镜表【关联剧本】列中找到对应镜头。任何一句台词如果找不到对应镜头编号 → 遗漏，立即补镜。
3. **角色轨迹检查**：每个有名字的角色，从"首次被提及/铺垫"→"正式出场"→"每个动作/台词"→"退场/转场"，整条轨迹在分镜表中连续可追踪，无断层。
4. **信息完整性**：剧本中任何关键信息（地点、时间、人物关系、关键物件），都在分镜中通过画面或对白得到了传达。没有剧本写了但分镜没体现的信息。

**审查清单（逐项打勾）**：
- [ ] 剧本每一个剧情节点 → 对应镜头编号 ______
- [ ] 剧本每一句台词/旁白 → 对应镜头编号 ______
- [ ] 每一个有名字的角色 → 完整的出场-行动-退场镜头链 ______
- [ ] 每一个场景的空间信息 → 已在分镜中建立 ______
- [ ] 剧本中没有一句对白、一个动作、一个信息点被遗漏

**但凡有一项打不上勾，不得交付。**

### 10.1 无声测试
将分镜序列静音播放，仅凭画面能否理解80%的故事、情绪、角色关系？不能 → 补镜头。

### 10.2 删除测试
把场景建立段落整个删掉，观众对后续剧情的理解是否受损？不受损 → 你的场景建立是可有无的明信片，重做。

### 10.3 情绪曲线检查
建立的情绪是否平滑过渡？有无意外断裂？是在帮助观众入戏还是在干扰？节奏曲线是否符合情绪弧线？

### 10.4 性价比检查
为了建立这个场景用了多少镜头、多少秒？时间和镜头成本是否换来了足够有价值的叙事效果？能否用更少的镜头达到80%效果？

### 10.5 导演意图终极反思
每个镜头选择、每次剪辑切换，背后是否都有一个不可替代的服务于最终故事表达的明确导演意图？能否为每个选择进行清晰辩护？

### 10.6 一致性检查
- 同一角色在不同镜头中外貌描述一致
- 场景光影/色调在连续镜头中连贯
- 180度轴线不跳轴
- 衔接逻辑链条完整，无逻辑断裂
- 视听信息不冲突（画面情绪与音效/配乐匹配）
