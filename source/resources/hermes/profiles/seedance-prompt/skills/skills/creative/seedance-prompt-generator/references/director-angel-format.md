# Director Angel 编译格式 — 完整字段说明

Seedance 2.0 视频生成的实际 API 请求通过 Director Angel 编译。以下为完整字段逐项说明。

## 顶层结构

### 提交级视觉风格硬规则

格式：`最终视频必须使用项目视觉风格：{风格描述}`

这是整个视频的最高层级美术指令，优先级高于所有其他视觉参考（包括 Board 参考图和角色参考图）。
Board 参考图只提供**构图参考**（角色位置、朝向、相对比例、场景空间关系），不得改变风格规则中的任何美术设定。

### Director Angel guidance

提示词框架声明，无实际内容。

### Base compiled prompt

逐镜头的画面描述 + 镜头语言 + 动作推进。这部分来自分镜的核心内容。

**与 buildGroupPrompt 的本质区别：**
- buildGroupPrompt：纯机械组装（cameraMovement, shotSize, videoPrompt 拼接）
- Director Angel：导演级编译（增加出镜角色、镜头职责、表演节点、叙事职责、组织要求）

### 导演校准要点

| 字段 | 说明 |
|------|------|
| 组间承接策略 | 当前组与上/下组的关系（独立提交/顺承/呼应） |
| 动作因果 | 前镜动作的后果如何延续到本镜 |
| 同一空间约束 | 同场景镜头必须共享空间参数（尺寸、光线方向、道具位置） |

### 组间承接建议

固定为 `当前组默认独立提交`。

### 单镜出镜控制

每镜明确列出角色。格式：`镜头N: 角色A（主）; 镜头N: 角色B（次）`

- `无角色/环境镜` → 不允许出现任何人物
- `角色名（主）` → 此角色是该镜头的核心焦点
- 角色首次出场时附带关键外貌锚点

### 标准分组连续构图锁定

仅当组内包含 2+ 个连续镜头且共享同一场景时启用。

| 字段 | 说明 |
|------|------|
| 构图基准 | 镜头2继承镜头1的摄像头位置、FOV |
| 硬锁位 | 目标角色在画面中的九宫格位置（如"右下三分点"） |
| 轴线规则 | 对话轴线 180° 不跨轴；动作轴线方向固定 |
| 站位摘要 | 角色相对位置（左/右/前/后） |

### 片段级改编事实层

4 个强制段落：

1. **【空间建立】** — 场景描述 + 关键道具 + 起始状态
2. **【动作爆发】** — 冲突/转折点 + 核心事件
3. **连续性控制** — 跨镜头空间/光影/角色一致性
4. **【结果落点】** — 片段结尾状态 + 余韵

## 逐镜头字段

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| 出镜角色 | string | 角色名或"无角色/环境镜" | 林星野（主）；苏晓（次） |
| 镜头语言 | object | 景别 + 运镜 + 特殊技法 | 景别 MS；运镜 跟拍；特殊技法 J-Cut |
| 镜头职责 | enum | 动作起势/氛围铺垫/情绪爆发/信息交代/转折过渡/余韵收束 | 情绪爆发 |
| 表演节点 | string | 具体动作/表情/肢体语言 | 紧握拳头，指节发白，眼眶泛红但不能流泪 |
| 动作推进 | string | 从上一状态到下一状态的转变 | 从凝视枪口→缓缓闭上眼睛→嘴角露出释然的微笑 |
| 叙事职责 | enum | action/动作起势/氛围铺垫/情绪爆发/信息交代/反应镜头 | 情绪爆发 |
| 组织要求 | enum + value | build/release/hold/sustain | 节奏 hold |
| 对白动作化 | string(可选) | 将台词转化为表演 | 说"我不会原谅你"→缓慢摇头，后退半步，眼神从愤怒转为空洞 |

### 镜头职责枚举

| 值 | 适用场景 |
|----|----------|
| 动作起势 | 动作的开始瞬间（拔枪、起跑、挥拳前摇） |
| 氛围铺垫 | 环境/气氛的建立镜头 |
| 情绪爆发 | 情感高潮瞬间 |
| 信息交代 | 关键信息/道具/线索的展示 |
| 转折过渡 | 情节转折的衔接镜头 |
| 余韵收束 | 片段结尾的收束镜头 |

### 组织要求枚举

| 值 | 含义 |
|----|------|
| build | 节奏加快/张力上升 |
| release | 节奏放缓/释放张力 |
| hold | 节奏维持/蓄力 |
| sustain | 节奏持续/保持状态 |

## 执行参数

```
Execution profile: Seedance 2.0 · {叙事模式} --resolution {分辨率} --ratio {画幅} --duration {时长s} --camera_fixed {true|false}
```

| 参数 | 选项 | 说明 |
|------|------|------|
| 叙事模式 | 标准叙事/动作叙事/氛围叙事 | |
| 分辨率 | 720p/1080p | 默认 720p |
| 画幅 | 16:9/9:16/1:1/4:3 | 默认 16:9 |
| 时长 | 整数秒 | 5-15 秒，如果组内总时长不在范围内需拆组 |
| camera_fixed | true/false | 全组是否固定机位 |

## 完整示例

```
提交级视觉风格硬规则：
最终视频必须使用项目视觉风格：新海诚风格、电影感、高质量、4K、HDR调色、柔光滤镜
该视觉风格优先级高于 Board 参考图。Board 参考图仅提供构图参考（角色位置/朝向/相对比例/场景空间关系），
不得改变上述视觉风格中的任何美术设定（画风/光影/色调/质感/角色造型/场景细节）。

Director Angel guidance:

Base compiled prompt:
废弃工厂内部，黄昏光线从破损的窗户斜射而入。林星野（红发、金瞳）站在生锈的机器前，
紧握拳头，指节发白。苏晓从阴影中走出，手中拿着档案袋。

导演校准要点：
- 组间承接策略：独立提交
- 动作因果：镜头1的空间建立→镜头2的角色引入→镜头3的冲突爆发
- 同一空间约束：全镜共享废弃工厂场景，光线方向统一为右上方45°

组间承接建议：当前组默认独立提交

【单镜出镜控制】：
- 出镜表：镜头1: 无角色/环境镜; 镜头2: 林星野（主）-红发金瞳; 镜头3: 林星野; 苏晓-黑发黑衣

【标准分组连续构图锁定】：
- 构图基准：镜头2继承镜头1的摄像头位置，镜头1推近→镜头2转为角色焦点
- 硬锁位：镜头2-3林星野在画面左侧三分点，镜头3苏晓从右侧入画
- 轴线规则：动作轴线从左到右
- 站位摘要：林星野在左（距机器2m），苏晓从右后方进入（距林3m）

片段级改编事实层：
1. 【空间建立】废弃工厂 + 黄昏光线 + 生锈机器 → 镜头1起始状态
2. 【动作爆发】苏晓出现 + 档案袋 → 镜头3核心事件
3. 连续性控制：光线方向统一（右上方45°），工厂空间比例一致，生锈机器作为背景锚点不变
4. 【结果落点】林星野接过档案袋手在颤抖 → 镜头3余韵

镜头1 [0s-3s]「废弃工厂」：
  出镜角色: 无角色/环境镜
  镜头语言: 景别 ELS；运镜 固定机位；特殊技法 无
  镜头职责: 氛围铺垫
  表演节点: 无
  动作推进: 黄昏光线渐暗 + 灰尘在光柱中飘浮
  叙事职责: 氛围铺垫
  组织要求: 节奏 hold

镜头2 [3s-7s]「林星野等待」：
  出镜角色: 林星野（主）- 红发、金瞳、黑色作战服
  镜头语言: 景别 MS；运镜 推拉（缓慢推进）；特殊技法 浅景深
  镜头职责: 动作起势
  表演节点: 紧握拳头，指节发白，低头盯着地面
  动作推进: 从等待状态→感知到有人靠近→微微抬头
  叙事职责: 动作起势
  组织要求: 节奏 build

镜头3 [7s-12s]「苏晓出现」：
  出镜角色: 林星野（主）-红发金瞳；苏晓（次）-黑发黑衣、手拿档案袋
  镜头语言: 景别 MLS；运镜 跟拍；特殊技法 无
  镜头职责: 情绪爆发
  表演节点: 苏晓从阴影中走出，林星野抬头，两人对视，苏晓递出档案袋，林星野接过，手在颤抖
  动作推进: 苏晓从阴影走出→递档案袋→林星野接过→手指颤抖
  叙事职责: 情绪爆发
  组织要求: 节奏 release
  对白动作化：林星野接过档案袋时，嘴唇微动但没说出话（替代"这是什么"台词）

Execution profile: Seedance 2.0 · 标准叙事 --resolution 720p --ratio 16:9 --duration 12 --camera_fixed false
```
