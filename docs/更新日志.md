# 更新日志

## v1.1.1 (2026-07-10) — 校验放宽 + 更新源限速绕过

### 修复
- **保存设置不再卡 api_key 缺失**:v1.1.0 强制要求 active config 字段填齐才能保存,v1.1.1 把校验下放到调用时,允许分批填
- **「检查更新」不再因 GitHub rate limit 失败**:60/h/IP 限速桶容易被打满,改成优先 `raw.githubusercontent.com/.../release/update.json`(无限速),失败再走 GitHub API

### 改动
- `ui/settings_dialog.py` 移除 active config 字段必填校验
- `core/updater.py` 新增 `fetch_update_json()` 函数 + 双源回退
- 版本号 1.1.0 → 1.1.1

## v1.1.0 (2026-07-09) — 一键更新

### 新增
- **一键自动更新**:菜单「帮助 → 检查更新」或点红点「🔴 新版 vX.X.X」
  - 后台流式下载 Setup.exe(带进度条 + 取消)
  - 下载完自动启动安装器静默装
  - 装完自动重启新版本
  - 整个过程无需用户手动下载 / 关软件
- `UpdateInfo` 加 `asset_url` + `asset_size` 字段
- 新增 `UpdateDownloader`(基于 QThread + urllib 流式)

### 改动
- Inno Setup 加 `CloseApplicationsFilter` / `RestartApplications` / `SetupMutex` 让安装器自动识别并关闭运行中的旧版本
- `main.py` / `漫剧助手X-2.iss` 版本号 1.0.3 → 1.1.0
