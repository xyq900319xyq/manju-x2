## v1.1.3 (2026-07-10) — 一键更新下载 UnicodeEncodeError 修复

### 修复
- **一键更新下载 Setup.exe 时报 `UnicodeEncodeError: 'ascii' codec can't encode characters in position 51-54`**:`update.json` 的 `url` 字段是 `.../download/1.1.2/漫剧助手X-2_v1.1.2_Setup.exe`,Setup.exe 文件名含中文"漫剧助手",`urllib.request.Request` → `http.client.putrequest` 内部 `request.encode('iso-8859-1')` 抛 UnicodeEncodeError,下载失败弹错

### 改动
- `core/updater.py` `_DownloadWorker.__init__` percent-encode url(中文 → `%E6%BC%AB...`),保证 `urlopen` 拿到全 ASCII url
- `build_x2.py` 写 `update.json` 时 `url` 字段也对文件名 percent-encode;`changelog_url` 改纯英文 `releases` 页(原 `docs/更新日志.md` 中文路径会让 QMessageBox 弹框也爆)
- `ui/main_window.py` `_on_error` 防御:`info.html_url` 段强制 ascii safe(中文 → `?`),即使将来还有别的中文路径也不爆
- `main.py` 启动时 `sys.stdout/stderr.reconfigure(encoding="utf-8", errors="replace")`,防 EXE 启动时 codepage 437/936 触发 `log` 输出 UnicodeEncodeError
- 版本号 1.1.2 → 1.1.3
