# 漫剧助手X-2 常见问题 (FAQ)

## 安装 / 升级

### Q: 安装需要管理员权限吗?
**A**: **不需要**。安装器使用 `PrivilegesRequired=lowest`,普通用户双击就能装。默认装到 `C:\漫剧助手X-2\`,如果装到 `Program Files\` 才会触发 UAC。

### Q: 安装时报"另一个安装正在进行"怎么破?
**A**: 等之前的安装完成,或者结束 `msiexec.exe` / 之前那个 Setup.exe 进程再试。

### Q: 装好后找不到软件图标?
**A**:
- 开始菜单搜 "漫剧助手"
- 或直接到 `C:\漫剧助手X-2\漫剧助手X-2.exe` 双击

### Q: 能装到 D 盘吗?
**A**: 可以,但要手动改安装目录。**注意**:装到非系统盘不会自动跟随用户漫游(重装系统后数据丢失)。

### Q: 装旧版装到一半,失败后无法重新装?
**A**:
1. 控制面板卸载残留
2. 手动删 `C:\漫剧助手X-2\`
3. 重新装

### Q: 升级会丢项目数据吗?
**A**: **不会**。覆盖安装只更新程序文件,`config\` / `data\` / `outputs\` 全部保留。

## API key / 配置

### Q: 没有 API key 能用吗?
**A**: 能启动软件和浏览 UI,但**所有生成功能**(剧本/分镜/生图/视频)都会报"未配置 API key"。

### Q: API key 存哪里?安全吗?
**A**:
- 存到 `config\secrets.bin`,**Windows DPAPI 加密**(绑定当前 Windows 用户账号)
- 其他用户登录你电脑也读不出来(Windows 账号隔离)
- 重装系统 → 密钥丢失,需重填
- 强烈建议:别把 `secrets.bin` 拷给别人,别截图发群里

### Q: 换电脑/重装系统后 API key 怎么办?
**A**: 重填。DPAPI 加密绑 Windows 用户账号,新环境无法解密。

### Q: 多个项目用不同 API 配置?
**A**: 支持。在 wizard 填多个 config,设置里切换。

### Q: 怎么知道哪个 API 余额?
**A**: 软件不直连查余额(各家 API 不同)。自己去 API 官网查(DeepSeek/Agnes/创维中转)。

### Q: 创维中转/Agnes/DeepSeek 哪个好?
**A**:
- **DeepSeek**: 价格便宜($0.14/M input),中文好,V3.1 稳定
- **Agnes**: 多模型聚合,按量计费,创维中转便宜
- **创维中转**: 国内速度最快,模型多
- 推荐: 国内用创维中转/Agnes,海外用 DeepSeek 直连

## 使用问题

### Q: 生成分镜/视频提示词卡住怎么办?
**A**:
1. 看 `logs\manju-最新.log`,搜 `❌` 或 `Error`
2. 90% 是 API key 无效/余额不足
3. 10% 是 prompt 太长 → 简化输入
4. 重启软件重试

### Q: 视频生成总是失败?
**A**:
1. 视频 API 需要单独配(LLM 生成分镜 + 视频 API 生成视频)
2. 创维中转和 Agnes 视频接口不同
3. 参生图(参考图)需要 imgbb key(图床上传)
4. 视频生成是异步的,看 logs 等几分钟到几十分钟

### Q: 资产图(角色/场景)怎么重新生成?
**A**:
- 资产页 → 选资产 → 点 "🔄 重新生成" → 弹对话框 → 选模型 → 生成
- **不**会自动覆盖原图,新图会单独存,你可以选 "替换" 或 "新建资产"

### Q: 输出文件在哪?
**A**: `outputs\<项目名>\` 下:
- `assets\<资产名>\<资产名>_<时间戳>.png` — 资产图
- `项目名--第N集--分镜词.txt` — 分镜文本
- `项目名--第N集--提示词.txt` — 视频提示词
- `🎞️ Segment N_<标题>_<时间戳>.mp4` — 视频文件

### Q: 输出文件名可以自定义吗?
**A**: 当前版本按 [记忆] 规则固定命名,见 `docs\API配置说明.md`。

## 性能 / 速度

### Q: 生成分镜要多久?
**A**:
- 简单剧情(5 集 × 5 分镜):1-2 分钟
- 复杂剧情(30 集 × 30 分镜):10-30 分钟
- DeepSeek V3.1 通常 < 创维中转
- API 超时/限流时会慢

### Q: 视频生成要多久?
**A**:
- 创维中转 1 段 5-15 秒:1-3 分钟
- Agnes: 1-5 分钟
- 复杂镜头(多参考图):5-10 分钟
- 排队时更长

### Q: 启动慢?
**A**:
- 首次启动:解压资源 + 初始化 DB,**30-60 秒**正常
- 二次启动:5-10 秒
- 如果二次启动也慢 → 看 logs,可能是 hermes 初始化失败

## 升级 / 更新

### Q: 软件提示更新但点不动?
**A**:
1. 检查网络(可能 GitHub 被墙)
2. 设置 → 高级 → 清除更新缓存(`.update_check_cache.json`)
3. 重启软件再试
4. 不行就手动去 GitHub 下载

### Q: 升级后配置丢了?
**A**: 不应该。覆盖安装保留 `config\`。如果丢了:
- 卸载后没清 `config\` → 重新装回老版本
- 或手动从备份恢复 `config\hermes_api.json` + `config\secrets.bin`

### Q: 怎么回滚到老版本?
**A**:
1. GitHub Releases → 找老版本(如 v0.9.0)→ 下载 Setup.exe
2. 双击装到原目录(覆盖)

### Q: 升级会清 outputs?
**A**: **不会**。`outputs\` 完全保留。

## 卸载 / 数据

### Q: 卸载会删项目数据吗?
**A**: **不会**。卸载只删 `C:\漫剧助手X-2\` 程序目录。`config\` / `data\` / `outputs\` / `logs\` 保留。

### Q: 怎么彻底清?
**A**:
```powershell
# 1. 先控制面板卸载
# 2. 手动删
Remove-Item -Recurse -Force "C:\漫剧助手X-2"
# 3. (可选) 删用户数据
Remove-Item -Recurse -Force "$env:APPDATA\manju-x2"  # 如果有
```

### Q: 数据能备份吗?
**A**: 备份 `outputs\` + `data\` 两个目录就行。`config\` 里的 secrets.bin 备份了重装系统也用不了(DPAPI 绑定用户)。

## 技术 / 错误信息

### Q: `DPAPI 解密失败` 报错?
**A**:
- 当前 Windows 用户变了(换账号登录)
- 系统重装过
- secrets.bin 损坏
- **解决**: 设置 → 重新填 API key

### Q: `hermes chat failed: ...` 报错?
**A**:
1. `logs\hermes-最新.log` 看详细错误
2. 90% 是 hermes 配置问题 → 设置 → 重新配 LLM API
3. 10% 是 hermes.exe 损坏 → 重装软件

### Q: `ModuleNotFoundError: No module named 'PySide6'`?
**A**: 这是开发环境错误。**用户版不应该出现**(PyInstaller 把所有依赖打包了)。如果出现说明 PyInstaller 打包漏了模块。

### Q: 杀软误报怎么办?
**A**:
- 360 / 火绒: 添加 `C:\漫剧助手X-2\` 到信任区
- Windows Defender: 弹窗选 "更多信息" → "仍要运行"
- 卡巴斯基 / Norton: 暂时关闭实时防护
- **不**要删除杀软(可能中毒)

## 反馈 / 报告 Bug

### Q: 怎么反馈 Bug?
**A**:
- GitHub Issues: https://github.com/&lt;your-org&gt;/manju-x2/issues
- 必带:`logs\manju-最新.log`(出错前最后 200 行)
- 必带:复现步骤
- 建议带:系统版本(Win10 22H2 / Win11 23H2 等)

### Q: 多久会修?
**A**:
- 阻塞性 bug(启动/安装/数据丢失):24-48 小时
- 一般 bug:1-2 周
- 功能建议:看反馈量,排期

### Q: 提需求会被采纳吗?
**A**: Issues 里 👍 多的优先。看 dev 端 `D:\漫剧助手\` 是不是已经在做。
